# 🚢 Titanic Survival Prediction App

A Machine Learning-based web application built using **Python**, **Scikit-learn**, and **Streamlit** that predicts whether a passenger would survive the Titanic disaster based on passenger details such as age, gender, class, fare, and family information.

This project demonstrates an end-to-end Machine Learning workflow including data preprocessing, feature engineering, model training, evaluation, and deployment.
